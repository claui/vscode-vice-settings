# VICE Settings

Keymaps, hotkeys and other settings for VICE

## What it does

This extension provides syntax highlighting for:

- GTK-3 hotkey definition files for VICE (`gtk3-*.vhk`)

- SDL hotkey definition files for VICE (`sdl-*.vhk`)
